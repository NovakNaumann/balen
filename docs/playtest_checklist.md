# Balen Playtest Checklist

This checklist is copied into the repository early so every phase can be measured against the playable-test target.

## Phase 0 Checklist

- [x] Current authority document is present.
- [x] Canon bible is present.
- [x] Earlier testbed plan is present.
- [x] Source manifest is present.
- [x] Latest attached Voyage JSON is copied unchanged.
- [x] Godot version is pinned in `README.md` and `project.godot`.
- [x] Testbed namespace rule is documented.
- [x] Explicit non-goals are documented.
- [x] Decisions, assumptions, and known issues files exist.

## External Playtest Target

- [ ] Start a new playable test.
- [ ] Move with WASD.
- [ ] Click across the plaza to move.
- [ ] Interrupt click movement with WASD.
- [ ] Rotate and zoom the camera.
- [ ] Switch party leader.
- [ ] Speak with Maelin.
- [ ] Identify all six plaza exits.
- [ ] Inspect the forged permit and other clues.
- [ ] Begin the crisis.
- [ ] Move onto and off elevated tiles.
- [ ] Use cover.
- [ ] Trigger one Reaction.
- [ ] Close or interact with the stable gate.
- [ ] Attempt to calm or restrain the Trotter.
- [ ] Attempt to capture or drive off the smugglers.
- [ ] Complete the scenario.
- [ ] Save, quit, and reload.
- [ ] Enter the developer laboratory.
- [ ] Compare humanoid and dragon footprints.
